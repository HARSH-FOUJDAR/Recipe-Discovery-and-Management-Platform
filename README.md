# Recipe Discovery and Management Platform

A modern recipe-sharing platform where users can discover, search, and manage recipes through an intuitive and responsive interface.

## Live Demo

🔗 https://remarkable-haupia-d108cb.netlify.app

## Features

* Browse and discover recipes
* Search and filter recipes
* Responsive design
* Interactive UI
* Smooth animations
* REST API integration

## Tech Stack

* React.js
* Node.Js
* Express.js
* MongoDB
* JavaScript
* Tailwind CSS
* Framer Motion
* REST APIs
